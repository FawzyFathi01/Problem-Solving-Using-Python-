## **Link Problem** : https://codeforces.com/contest/886/problem/A


## **Code**

```python
import sys
scores = list(map(int,input().split()))
com = sum(scores)/2
scores = sorted(scores,reverse=True)
if com == 0:
    print("YES")
    sys.exit()
ans = 0
for i in range(4):
    if ans + scores[i] < com:
        ans += scores[i]
    for j in range(i+1,5):
        if ans + scores[j] < com:
            ans += scores[j]
            break
    for k in range(j+1,6):
        if ans + scores[k] == com:
            ans+= scores[k]
            print("YES")
            sys.exit()
    ans = 0
print("NO")
```
