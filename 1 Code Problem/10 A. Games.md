## **Link Problem** : https://codeforces.com/contest/268/problem/A


## **Code**

```python
n = int(input())
pairs = []
for i in range(n):
    x, y = map(int , input().split())
    pairs.append((x,y))
ans = 0
for i in range(n-1):
    for j in range(i+1,n):
        if pairs[i][0] == pairs[j][1]:
            ans+=1
        if pairs[i][1] == pairs[j][0]:
            ans+=1
print(ans)
```
