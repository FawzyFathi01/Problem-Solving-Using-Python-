## **Link Problem** : https://codeforces.com/problemset/problem/1760/C

## **Code**

```python
t = int(input())

for i in range (t):
    n = int (input())
    numbers = list (map (int,input().split()))
    maximum_1 = 0
    maximum_2 = 0
    for item in numbers:
        if item > maximum_1:
            maximum_1 = item

  

    cnt = 0
    for item in numbers:
        if item == maximum_1:
            cnt+=1
            continue
        if item > maximum_2:
            maximum_2 = item

  
  
    ans = []
    for item in numbers:
        if item < maximum_1:
            si = item - maximum_1
            ans.append(si)
        elif item == maximum_1 and cnt >= 2:
            si=0
            ans.append(si)
        elif item == maximum_1 and cnt == 1:
            si = maximum_1 - maximum_2
            ans.append(si)
    for item in ans:
        print(item, end=" ")
    print()
```
