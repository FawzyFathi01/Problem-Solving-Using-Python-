## **Code**

```python

2. cards = list(map(int, input().split()))

3. l = 0
4. r = size-1
5. s = 0
6. d = 0
7. loop = 0

8. if size%2 == 0:
9. loop = (size//2)
10. else:
11. loop = (size//2)+1

12. for i in range(loop):
13. if cards[l] > cards[r]:
14. s+= cards[l]
15. l+=1
16. else:
17. s+= cards[r]
18. r-=1

19. if size%2 != 0 and i == loop-1:
20. break

21. if cards[l] > cards[r]:
22. d+= cards[l]
23. l+=1
24. else:
25. d+= cards[r]
26. r-=1

27. print(f"{s} {d}")
```
