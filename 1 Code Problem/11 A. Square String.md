## **Link Problem** : https://codeforces.com/problemset/problem/1619/A

## **Code**

```python
t = int(input())
for i in range(t):
    str = input()
    n = len(str)
    if n % 2 != 0:
        print("NO")
        continue
    mid_idx = n//2
    pre_str = str[:mid_idx]
    suf_str = str[mid_idx:]
    if pre_str == suf_str:
        print("YES")
    else:
        print("NO")
```
