## **Link Problem** : https://codeforces.com/problemset/problem/903/A

## **Code**

```python
t = int(input())
for i in range(t):
    x = int(input())
    nums_3 = x // 3
    remind = x % 3
    if remind == 0:
        print("YES")
    elif (remind == 1 and nums_3 >= 2) or (remind == 2 and nums_3 >= 4):
        print("YES")
    else:
        print("NO")
```
